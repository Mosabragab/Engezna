Usage Guidelines

Best practices for brand consistency

Logo Usage

✓
Do: Use full "إنجزنا" text

Always use the complete Arabic name for the logo.

✗
Don't: Use single letters

Never use "إ" or "E" alone for the main logo/icon.

✓
Do: Use Charcoal for text

Logo text should be #0F172A on light backgrounds.

✗
Don't: Use pure black

Never use #000000 - use Charcoal #0F172A instead.

Color Usage

✓
Do: Use tint backgrounds for cards

Apply 10% opacity tints with darker text for contrast.

✗
Don't: Use solid colors on stat cards

Avoid solid gradient backgrounds with white text on cards.

Typography

✓
Do: Use Aref Ruqaa for logo only

Use Noto Sans Arabic for all other Arabic text.

✗
Don't: Mix fonts incorrectly

Don't use Aref Ruqaa for body text or UI elements.
