Usage Guidelines

Best practices for brand consistency

Color Usage

✓
Do: Use tint backgrounds for stat cards

Apply 10% opacity tints with darker text variants for proper contrast.

✗
Don't: Use solid colors on stat cards

Avoid solid gradient backgrounds with white text on dashboard cards.

✓
Do: Use primary color for CTAs

Buttons and links should use the primary blue (#009DE0).

✗
Don't: Modify brand colors

Never alter the hue or saturation of brand colors.

Typography

✓
Do: Use Cairo for Arabic

Cairo is our primary Arabic font for all communications.

✗
Don't: Mix too many weights

Stick to 2-3 weights per design for visual harmony.

Logo

✓
Do: Maintain proportions

Always scale the logo proportionally.

✗
Don't: Stretch or rotate

Never distort, rotate, or add effects to the logo.
