## الأدوار الوظيفية

### 1. مدير الفريق

**المسمى:** City Operations Manager
**الراتب:** 7,000 ج.م/شهر
**التقارير إلى:** المؤسس

#### المسؤوليات

```
الإدارة:
• قيادة فريق المندوبين
• توزيع المهام والمناطق
• متابعة الأداء اليومي
• حل المشاكل والتصعيدات

العمليات:
• تنسيق عمليات Onboarding التجار
• إدارة الدعم الفني للتجار والعملاء
• التواصل مع التجار المهمين
• تقارير أسبوعية للإدارة

التسويق:
• تنفيذ الحملات المحلية
• التنسيق مع المؤثرين
• إدارة صفحات السوشيال ميديا المحلية
```

#### KPIs

| المؤشر | الهدف | الوزن |
|--------|-------|-------|
| عدد التجار النشطين | ≥ الهدف الشهري | 25% |
| رضا التجار | ≥ 4/5 | 20% |
| وقت حل المشاكل | ≤ 24 ساعة | 15% |
| أداء فريق المندوبين | ≥ 80% من الأهداف | 20% |
| جودة التقارير | اكتمال + دقة | 10% |
| الالتزام بالميزانية | ≤ 100% | 10% |

### 2. مندوب المبيعات

**المسمى:** Sales Representative
**الراتب:** 6,000 ج.م/شهر
**التقارير إلى:** مدير الفريق

#### المسؤوليات

```
المبيعات:
• زيارة التجار المحتملين
• تقديم عروض المنصة
• إتمام عملية التسجيل
• جمع الوثائق المطلوبة

Onboarding:
• مساعدة التجار في إعداد المتجر
• تدريب التجار على لوحة التحكم
• متابعة التجار الجدد

المتابعة:
• زيارات دورية للتجار النشطين
• حل المشاكل البسيطة
• جمع الملاحظات والاقتراحات
```

#### KPIs

| المؤشر | الهدف | الوزن |
|--------|-------|-------|
| تجار جدد/أسبوع | ≥ 5 تجار | 30% |
| معدل التحويل | ≥ 40% | 20% |
| تجار مُدرَّبون | ≥ 90% من المسجلين | 20% |
| زيارات يومية | ≥ 8 زيارات | 15% |
| رضا التجار | ≥ 4/5 | 15% |

#### الجدول اليومي

```
08:00-09:00  │ اجتماع صباحي + مراجعة الخطة
09:00-13:00  │ زيارات ميدانية (5-6 تجار)
13:00-14:00  │ استراحة + توثيق
14:00-17:00  │ زيارات ميدانية (4-5 تجار)
17:00-18:00  │ متابعة WhatsApp
18:00-19:00  │ نشر محتوى + رد على التعليقات
19:00-20:00  │ تقرير يومي + تخطيط الغد
```

## معايير التوظيف

### مدير الفريق

#### المتطلبات الأساسية

| المعيار | المطلوب |
|---------|---------|
| التعليم | بكالوريوس (يفضل إدارة أعمال/تسويق) |
| الخبرة | 2-4 سنوات في المبيعات أو العمليات |
| المهارات | قيادة الفريق، حل المشاكل، التواصل |
| اللغة | العربية (ممتاز)، الإنجليزية (جيد) |
| الموقع | مقيم في المحافظة أو مستعد للانتقال |

#### المهارات المطلوبة

```
أساسية:
✅ خبرة في إدارة فرق صغيرة
✅ معرفة بالسوق المحلي
✅ مهارات تواصل ممتازة
✅ القدرة على العمل تحت الضغط
✅ استخدام الكمبيوتر والتطبيقات

مفضلة:
⭐ خبرة في التجارة الإلكترونية
⭐ خبرة في قطاع التوصيل/المطاعم
⭐ شبكة علاقات محلية
```

### مندوب المبيعات

| المعيار | المطلوب |
|---------|---------|

#### المهارات المطلوبة

```
أساسية:
✅ شخصية اجتماعية ومبادرة
✅ القدرة على الإقناع
✅ الالتزام بالمواعيد
✅ استخدام الهاتف الذكي بكفاءة
✅ معرفة بالمنطقة الجغرافية

مفضلة:
⭐ خبرة سابقة في المبيعات
⭐ معرفة بالتجار المحليين
⭐ نشاط على السوشيال ميديا
```

## عملية التوظيف

### المراحل

```
1. استقبال السير الذاتية
   ↓
2. فرز أولي (السيرة الذاتية)
   ↓
3. مقابلة هاتفية (15 دقيقة)
   ↓
4. مقابلة شخصية (30-45 دقيقة)
   ↓
5. اختبار عملي (لو لزم)
   ↓
6. القرار النهائي
   ↓
7. عرض العمل
   ↓
8. التدريب (أسبوع)
```

### أسئلة المقابلة

#### مدير الفريق

```
1. أخبرنا عن تجربتك في قيادة فريق؟
2. كيف تتعامل مع موظف أداؤه ضعيف؟
3. ما معرفتك بسوق التوصيل في مصر؟
4. كيف ستقنع تاجر متردد بالانضمام؟
5. أعطنا مثال على مشكلة حللتها في عملك السابق؟
```

#### مندوب المبيعات

```
1. لماذا تريد العمل في المبيعات؟
2. كيف تتعامل مع الرفض من العملاء؟
3. ما معرفتك بالمنطقة التي ستعمل فيها؟
4. أقنعني بالاشتراك في إنجزنا (تمثيل دور)
5. ما هي أهدافك المهنية؟
```

## التدريب والتطوير

### برنامج التدريب الأولي (أسبوع)

| اليوم | المحتوى |
|-------|---------|
| 1 | تعريف بالشركة والمنتج |
| 2 | تدريب على التطبيق ولوحة التحكم |
| 3 | تدريب على Sales Script |
| 4 | تدريب ميداني (مع مدير) |
| 5 | تدريب ميداني (مستقل + تقييم) |

### التطوير المستمر

```
أسبوعياً:
• اجتماع فريق (مراجعة الأداء)
• مشاركة أفضل الممارسات

شهرياً:
• تدريب على مهارة جديدة
• مراجعة أداء فردية

ربع سنوياً:
• تقييم أداء شامل
• مراجعة الأهداف والرواتب
```
