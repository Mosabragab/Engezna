## نظرة عامة

**إنجزنا (Engezna)** هو سوق إلكتروني B2C يربط المتاجر المحلية بالعملاء في المحافظات المصرية.

| البند       | التفاصيل                      |
| ----------- | ----------------------------- |
| **النوع**   | Delivery Marketplace Platform |
| **السوق**   | مصر (البداية من صعيد مصر)     |
| **الإطلاق** | فبراير 2026                   |
| **التقدم**  | 97% MVP                       |

### الفئات المستهدفة

| الفئة                      | الوصف                             | وضع التشغيل |
| -------------------------- | --------------------------------- | ----------- |
| **المطاعم**                | مطاعم، كافيهات، وجبات سريعة       | Standard    |

| **السوبر ماركت**           | سوبر ماركت وبقالة                 | Hybrid      |
| **الخضراوات والفاكهه**     | محلات الخضار والفاكهة الطازجة     | Custom      |
| **البن والحلويات**         | محلات البن والمكسرات والحلويات    | Standard    |

## أوضاع التشغيل

### 1. Standard Mode (القائمة الثابتة)

```
للمطاعم، الصيدليات، البن والحلويات
• قائمة منتجات بأسعار ثابتة
• Variants (أحجام مختلفة)
• عروض وخصومات
```

### 2. Custom Mode (التسعير المخصص)

```
للخضراوات والفاكهه، الجزارة، الأسماك
• التاجر يحدد السعر حسب الوزن/الكمية
• نظام عروض أسعار
• مرونة كاملة
```

### 3. Hybrid Mode

```
للسوبر ماركت
• جمع بين الوضعين
• بعض المنتجات ثابتة
• بعض المنتجات بالوزن
```

## الميزة التنافسية التقنية

### Custom Mode

```
ميزة فريدة غير موجودة عند المنافسين

• التاجر يرسل عرض سعر للعميل
• العميل يقبل أو يرفض
• مثالي للمنتجات بالوزن
```

### Triple Broadcast (للطلبات المخصصة)

```
1. الطلب يُرسل لـ 3 تجار
2. أول من يرد يحصل على الطلب
3. يضمن سرعة الاستجابة
```

### نظام Variants

```
• أحجام مختلفة للمنتج
• أسعار مختلفة لكل حجم
• اختيار الافتراضي
```

## ملخص بالعربية

### نظرة عامة على المنتج

**إنجزنا** هو سوق إلكتروني للتوصيل يربط التجار المحليين بالعملاء في المحافظات المصرية. المنتج جاهز بنسبة 97% ومقرر إطلاقه في فبراير 2026.

### الفئات المستهدفة (5 فئات رئيسية)

1. **المطاعم** - مطاعم، كافيهات، وجبات سريعة (Standard Mode)
2. **الصيدليات** - صيدليات ومستلزمات طبية (Standard Mode)
3. **السوبر ماركت** - سوبر ماركت وبقالة (Hybrid Mode)
4. **الخضراوات والفاكهه** - محلات الخضار والفاكهة الطازجة (Custom Mode)
5. **البن والحلويات** - محلات البن والمكسرات والحلويات (Standard Mode)

### أوضاع التشغيل

| الوضع      | الاستخدام                                  | الوصف                      |
| ---------- | ----------------------------------------- | -------------------------- |
| Standard   | المطاعم، الصيدليات، البن والحلويات        | قائمة بأسعار ثابتة         |
| Custom     | الخضراوات والفاكهه، الجزارة، الأسماك      | تسعير بالوزن/عروض أسعار   |
| Hybrid     | السوبر ماركت                              | جمع بين الوضعين            |

### الميزات الأساسية

**للعملاء:**
- تصفح المتاجر والمنتجات
- سلة المشتريات والدفع (كاش أو إلكتروني)
- تتبع الطلب لحظيا
- التقييم والدردشة مع التاجر

**للتجار:**
- لوحة تحكم كاملة
- إدارة القائمة والطلبات
- تحليلات وتقارير
- نظام تسويات

**الميزة التنافسية التقنية:**
- Custom Mode للمنتجات بالوزن (فريد)
- Triple Broadcast للطلبات المخصصة
- نظام Variants للأحجام المختلفة

### التقنيات المستخدمة

Next.js 16، React 19، Supabase، Vercel، Kashier للدفع الإلكتروني
