## 3. مقارنة مع المنافسين

### جدول المقارنة الشامل

| البند | إنجزنا | طلبات | المنيوز |
|-------|--------|-------|---------|
| **رسوم التسجيل** | 0-2,000 ج.م | 3,000-15,000 ج.م | 2,000-5,000 ج.م |
| **العمولة** | 0-7% | 15-25% | 15-20% |
| **رسوم التصوير** | مجاناً | 500-1,500 ج.م | 500-1,000 ج.م |
| **رسوم الإعلانات** | اختيارية | إلزامية أحياناً | اختيارية |
| **فترة تجريبية** | 3 شهور مجاناً | لا | لا |

### حساب التوفير للتاجر (سنوياً)

```
مثال: مطعم متوسط
• 100 طلب/شهر
• متوسط الطلب: 150 ج.م
• إجمالي المبيعات: 15,000 ج.م/شهر

┌─────────────────────────────────────────────────────────────────┐
│                    المقارنة السنوية                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  طلبات (20% عمولة):                                             │
│  ─────────────────                                               │
│  • رسوم تسجيل: 3,000 ج.م                                        │
│  • عمولة سنوية: 15,000 × 20% × 12 = 36,000 ج.م                 │
│  • الإجمالي: 39,000 ج.م/سنة                                    │
│                                                                 │
│  إنجزنا (7% عمولة):                                             │
│  ────────────────                                                │
│  • رسوم تسجيل: 0-2,000 ج.م                                     │

│  • أول 3 شهور: 0 ج.م                                           │
│  • الإجمالي: ~11,450 ج.م/سنة                                   │
│                                                                 │
│  ════════════════════════════════════════════════════════════   │
```

## 6. سيناريوهات التسعير

### السيناريو 1: مطعم صغير

```
النوع: مطعم كشري
الطلبات: 50/شهر
متوسط الطلب: 80 ج.م

الموجة 1:
• شهر 1-3: 0 ج.م عمولة
• شهر 4+: 50 × 80 × 7% = 280 ج.م/شهر

المقارنة مع طلبات:
• 3,000 تسجيل + 50 × 80 × 20% × 12 = 12,600 ج.م/سنة
• إنجزنا: 280 × 9 = 2,520 ج.م/سنة
• توفير: 10,080 ج.م (80%)
```

### السيناريو 2: سوبر ماركت متوسط

```
النوع: سوبر ماركت
الطلبات: 200/شهر
متوسط الطلب: 250 ج.م

الموجة 2:
• رسوم تسجيل: 2,000 ج.م
• شهر 1-3: 0 ج.م عمولة
• شهر 4+: 200 × 250 × 7% = 3,500 ج.م/شهر

المقارنة مع طلبات:
• 5,000 تسجيل + 200 × 250 × 18% × 12 = 113,000 ج.م/سنة
• إنجزنا: 2,000 + 3,500 × 9 = 33,500 ج.م/سنة
• توفير: 79,500 ج.م (70%)
```

### السيناريو 3: جزار (Custom Mode)

```
النوع: جزارة
الطلبات: 80/شهر
متوسط الطلب: 400 ج.م

الموجة 1:
• شهر 1-3: 0 ج.م عمولة
• شهر 4+: 80 × 400 × 7% = 2,240 ج.م/شهر

ميزة Custom Mode:
• التاجر يحدد السعر حسب الوزن الفعلي
• لا أسعار ثابتة مسبقاً
• مرونة كاملة
```

## 8. إدارة الاعتراضات

### "ليه أدفع 2,000 ج.م رسوم تسجيل؟"

```
الرد:
─────
"هذه رسوم مرة واحدة فقط، وتشمل:
• إعداد متجرك بالكامل
• تصوير المنتجات (مجاناً)
• تدريب على المنصة
• دعم فني لمدة شهر

في طلبات، الرسوم 3,000 ج.م + تصوير 1,000 ج.م = 4,000 ج.م
معانا وفرت 50%"
```

### "7% لسه كتير"

```
الرد:
─────
"خلينا نحسبها:
• طلب 150 ج.م × 7% = 10.5 ج.م بس
• في طلبات: 150 × 20% = 30 ج.م

كل طلب بتوفر 19.5 ج.م
100 طلب/شهر = توفير 1,950 ج.م/شهر

وكمان أول 3 شهور بدون أي عمولة!"
```

### "مش عايز أدفع دلوقتي"

```
الرد:
─────
"تمام، عندنا خيارات:
1. الموجة الأولى: صفر تماماً (لو لسه فيها أماكن)
2. تقسيط: 1,000 دلوقتي + 1,000 بعد شهر
3. استنى الموجة الجاية لو فيها عرض أحسن

بس خلي بالك، كل ما بدري انضممت، كل ما استفدت أكتر"
```
