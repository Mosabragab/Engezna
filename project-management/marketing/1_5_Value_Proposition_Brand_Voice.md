## 🎨 الهوية البصرية والنفسية

### لماذا "إنجزنا"؟

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    معنى الاسم ودلالاته                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  📝 الاسم: إنجزنا (Engezna)                                                │
│                                                                             │
│  🔤 الأصل:                                                                 │
│  • من الفعل "أنجز" = أتم، أكمل، حقق                                       │
│  • "إنجزنا" = نحن ننجز لك، نخلص لك، نوفر عليك                             │
│                                                                             │
│  💡 الدلالات:                                                              │
│  ┌─────────────────────────────────────────────────────────────────┐       │
│  │  ✅ الإنجاز: "هننجزلك اللي محتاجه"                              │       │
│  │  ✅ السرعة: "هنخلصلك بسرعة"                                     │       │
│  │  ✅ الثقة: "اطمن، إحنا هننجزها"                                 │       │
│  │  ✅ الشراكة: "إنجزنا" = نحن + أنت = فريق                        │       │
│  │  ✅ المحلية: كلمة عامية مصرية مفهومة للجميع                     │       │
│  └─────────────────────────────────────────────────────────────────┘       │
│                                                                             │
│  🎯 الربط العاطفي:                                                         │
│  • "عايز تطلب؟" = السؤال (الحاجة)                                         │
│  • "إنجزنا" = الجواب (الحل)                                                │
│  • البساطة في السؤال والجواب = سهولة الاستخدام                             │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
