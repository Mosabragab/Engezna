## 2. North Star Metric | المؤشر الرئيسي

### GMV (Gross Merchandise Value)

```
┌─────────────────────────────────────────────────────────────────┐
│  ⭐ GMV = إجمالي قيمة الطلبات المكتملة                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  الصيغة:                                                        │
│  GMV = عدد الطلبات × متوسط قيمة الطلب (AOV)                    │
│                                                                 │
│  الأهداف:                                                       │
│  ─────────                                                       │
│  │ الشهر  │ الطلبات │ AOV    │ GMV          │                  │
│  │────────│─────────│────────│──────────────│                  │
│  │ 1      │ 200     │ 150    │ 30,000 ج.م   │                  │
│  │ 3      │ 800     │ 150    │ 120,000 ج.م  │                  │
│  │ 6      │ 3,000   │ 160    │ 480,000 ج.م  │                  │
│  │ 12     │ 10,000  │ 170    │ 1,700,000 ج.م│                  │
│                                                                 │
│  لماذا GMV؟                                                     │
│  ─────────                                                       │
│  • يعكس صحة المنصة ككل                                          │
│  • يجمع بين التجار والعملاء والعمليات                           │
│  • سهل الفهم والتتبع                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 3. مؤشرات التجار

### 3.1 مؤشرات الاكتساب

| المؤشر           | التعريف                | الهدف       | التكرار |
| ---------------- | ---------------------- | ----------- | ------- |
| **تجار جدد**     | عدد التجار المسجلين    | 15-20/أسبوع | أسبوعي  |
| **معدل التحويل** | المسجلين ÷ الزيارات    | ≥ 40%       | أسبوعي  |
| **CAC**          | تكلفة اكتساب التاجر    | ≤ 300 ج.م   | شهري    |
| **وقت التفعيل**  | من التسجيل للطلب الأول | ≤ 7 أيام    | أسبوعي  |

### 3.2 مؤشرات النشاط

| المؤشر                | التعريف                     | الهدف      | التكرار |
| --------------------- | --------------------------- | ---------- | ------- |
| **التجار النشطون**    | تجار بـ 3+ طلبات/أسبوع      | ≥ 70%      | أسبوعي  |
| **معدل الطلبات/تاجر** | طلبات ÷ تجار نشطين          | ≥ 20/أسبوع | أسبوعي  |
| **معدل القبول**       | الطلبات المقبولة ÷ الإجمالي | ≥ 95%      | يومي    |
| **وقت القبول**        | من الطلب للقبول             | ≤ 5 دقائق  | يومي    |

### 3.3 مؤشرات الاحتفاظ

| المؤشر                  | التعريف                   | الهدف    | التكرار  |
| ----------------------- | ------------------------- | -------- | -------- |
| **معدل الاحتفاظ**       | التجار النشطون بعد 30 يوم | ≥ 85%    | شهري     |
| **معدل الخروج (Churn)** | التجار غير النشطين        | ≤ 5%/شهر | شهري     |
| **NPS التجار**          | صافي نقاط الترويج         | ≥ 40     | ربع سنوي |

## 6. مؤشرات مالية

### 6.1 الإيرادات

| المؤشر                 | التعريف                    | الهدف       | التكرار |
| ---------------------- | -------------------------- | ----------- | ------- |
| **GMV**                | إجمالي قيمة الطلبات        | نمو 30%/شهر | شهري    |
| **إيرادات العمولة**    | GMV × 7%                   | -           | شهري    |
| **رسوم التسجيل**       | عدد التجار × الرسوم        | -           | شهري    |
| **معدل نمو الإيرادات** | (الحالي - السابق) ÷ السابق | ≥ 20%/شهر   | شهري    |

### 6.2 التكاليف

| المؤشر                | التعريف                      | الهدف        | التكرار |
| --------------------- | ---------------------------- | ------------ | ------- |
| **التكاليف الثابتة**  | الرواتب + التقنية            | ≤ 39,000 ج.م | شهري    |
| **التكاليف المتغيرة** | التسويق + الحوافز            | ≤ 10% من GMV | شهري    |
| **CAC المدمج**        | (تسويق + مبيعات) ÷ عملاء جدد | ≤ 50 ج.م     | شهري    |
| **معدل الحرق**        | النفقات - الإيرادات          | تناقصي       | شهري    |

### 6.3 الربحية

| المؤشر                  | التعريف                             | الهدف             | التكرار  |
| ----------------------- | ----------------------------------- | ----------------- | -------- |
| **هامش الربح الإجمالي** | (إيرادات - تكاليف مباشرة) ÷ إيرادات | ≥ 80%             | شهري     |
| **صافي الربح**          | إيرادات - كل التكاليف               | إيجابي من الشهر 5 | شهري     |
| **LTV/CAC**             | قيمة العميل ÷ تكلفة الاكتساب        | ≥ 3x              | ربع سنوي |
| **Payback Period**      | CAC ÷ الإيراد/عميل/شهر              | ≤ 6 شهور          | ربع سنوي |

## ملخص بالعربية

هذا الملف يوضح المؤشرات ومقاييس الأداء لمنصة إنجزنا. النقاط الرئيسية:

**المؤشر الرئيسي (North Star): GMV**

- إجمالي قيمة الطلبات المكتملة
- الهدف: 1,700,000 ج.م بنهاية السنة الأولى

**مؤشرات التجار:**

- تجار جدد: 15-20/أسبوع
- معدل التحويل: ≥ 40%
- معدل الاحتفاظ: ≥ 85%
- NPS التجار: ≥ 40

**مؤشرات العملاء:**

- عملاء جدد: 50-100/أسبوع
- CAC: ≤ 30 ج.م
- معدل التكرار: ≥ 40%
- LTV: ≥ 500 ج.م

**مؤشرات العمليات:**

- معدل الإكمال: ≥ 95%
- وقت التوصيل: ≤ 45 دقيقة
- رضا العملاء: ≥ 85%

**أهداف المراحل:**

- نهاية أبريل (شهر 1): 50-60 تاجر
- نهاية يوليو (شهر 4): 100-150 تاجر + 800-1,200 عميل
- نهاية أكتوبر (شهر 7): 200+ تاجر في محافظتين

**الفئات المستهدفة:** المطاعم، الصيدليات، السوبر ماركت، الخضراوات والفاكهه، البن والحلويات
