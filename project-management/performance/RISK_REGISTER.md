## 1. إطار إدارة المخاطر

### مصفوفة تقييم المخاطر

```
┌─────────────────────────────────────────────────────────────────┐
│                    مصفوفة المخاطر                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  الاحتمالية ▲                                                   │
│             │                                                   │
│    عالي    │   متوسط    │    عالي     │   حرج      │          │
│    (4-5)   │   (8-12)   │   (12-15)   │  (16-25)   │          │
│             │            │             │            │          │
│    متوسط   │   منخفض    │   متوسط    │   عالي     │          │
│    (3)     │   (3-6)    │   (6-9)    │  (9-15)    │          │
│             │            │             │            │          │
│    منخفض   │   منخفض    │   منخفض    │   متوسط   │          │
│    (1-2)   │   (1-4)    │   (2-6)    │   (4-10)   │          │
│             │            │             │            │          │
│             └────────────┴─────────────┴────────────► التأثير   │
│                 منخفض       متوسط         عالي                 │
│                 (1-2)       (3)          (4-5)                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

درجة الخطر = الاحتمالية × التأثير

🔴 حرج (16-25): تصعيد فوري + خطة طوارئ
🟠 عالي (9-15): مراقبة يومية + خطة تخفيف
🟡 متوسط (4-8): مراقبة أسبوعية
🟢 منخفض (1-3): مراقبة شهرية
```

## ملخص بالعربية

هذا الملف يوضح سجل المخاطر لمنصة إنجزنا. النقاط الرئيسية:

**إطار تقييم المخاطر:**
- درجة الخطر = الاحتمالية × التأثير
- حرج (16-25): تصعيد فوري
- عالي (9-15): مراقبة يومية
- متوسط (4-8): مراقبة أسبوعية
- منخفض (1-3): مراقبة شهرية

**أعلى المخاطر:**
1. **R01 - فشل اكتساب التجار (15):** خطط تخفيف تشمل تدريب مكثف وعروض الموجة الأولى
2. **R02 - تسرب التجار (12):** متابعة دورية وبرنامج ولاء
3. **R03 - ضعف اكتساب العملاء (12):** حملات سوشيال ميديا وبرنامج إحالة
4. **R04 - نفاد السيولة (10):** احتياطي 3 أشهر
5. **R06 - تعطل المنصة (10):** نسخ احتياطية ومراقبة 24/7
6. **R08 - دخول طلبات (10):** بناء قاعدة قوية بسرعة

**جدول المراجعة:**
- يومي: المخاطر التشغيلية
- أسبوعي: كل المخاطر العالية
- شهري: كل المخاطر
- ربع سنوي: المخاطر الخارجية

**الفئات المستهدفة:** المطاعم، الصيدليات، السوبر ماركت، الخضراوات والفاكهه، البن والحلويات
