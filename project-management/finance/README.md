## نقطة التعادل (Break-even)

### الافتراضات

- 150 تاجر نشط
- 100 طلب/تاجر/شهر
- متوسط الطلب: 150 ج.م
- عمولة: 7%
- فترة صفر عمولة: 3 شهور

### الحساب

```
الإيرادات الشهرية (بعد 3 شهور):
150 تاجر × 100 طلب × 150 ج.م × 7% = 157,500 ج.م

التكاليف الشهرية: 39,000 ج.م

صافي الربح: 157,500 - 39,000 = 118,500 ج.م

نقطة التعادل: الشهر 4-5
```
