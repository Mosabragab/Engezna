## مراحل الـ Onboarding

### المرحلة 1: التواصل الأولي (يوم 1)

```
□ تحديد موعد زيارة مع التاجر
□ تجهيز مواد العرض (بروشور، بطاقة عمل)
□ التأكد من فهم عرض الموجة الحالية
```

### المرحلة 2: الزيارة والعرض (يوم 1-2)

```
□ شرح منصة إنجزنا ومميزاتها
□ المقارنة مع المنافسين (طلبات: 3,000 ج.م + 18-25% vs إنجزنا: 0 + 0-7%)
□ شرح نظام التسعير (Standard, Custom, Hybrid)
□ الإجابة على استفسارات التاجر
□ الحصول على موافقة مبدئية
```

### المرحلة 3: التسجيل (يوم 2-3)

```
□ جمع البيانات الأساسية:
  □ اسم المتجر (عربي + إنجليزي)
  □ رقم الهاتف
  □ العنوان + الإحداثيات GPS
  □ الفئة (مطاعم، سوبرماركت، صيدلية، خضار وفاكهة، أخرى)
  □ نظام التشغيل المفضل (قائمة، تسعير، هجين)

□ جمع الوثائق:
  □ صورة البطاقة الشخصية
  □ صورة السجل التجاري (لو متاح)
  □ صور المتجر (واجهة + داخلي)
  □ شعار المتجر (لو متاح)

□ إنشاء الحساب على المنصة
□ تفعيل المحافظة والمدينة
```

### المرحلة 4: إعداد القائمة (يوم 3-5)

```
□ شرح طرق إضافة المنتجات:
  □ إضافة يدوية من لوحة التحكم
  □ استيراد من Excel (الطريقة الأسرع)

□ مساعدة التاجر في:
  □ تصنيف المنتجات
  □ تحديد الأسعار
  □ إضافة الصور
  □ إعداد المتغيرات (Variants) لو لزم

□ مراجعة القائمة والتأكد من اكتمالها
```

### المرحلة 5: التدريب (يوم 5-7)

```
□ تدريب على لوحة التحكم:
  □ استقبال الطلبات
  □ تأكيد/رفض الطلبات
  □ تحديث حالة الطلب
  □ التواصل مع العميل (Chat)

□ تدريب على نظام التسعير (لو يستخدم Custom Mode):
  □ استقبال طلبات التسعير
  □ تسعير المنتجات
  □ إرسال العرض للعميل

□ شرح نظام التسويات:
  □ فترات التسوية
  □ العمولة والحسابات
  □ طرق الدفع

□ شرح التقييمات والمراجعات
```

### المرحلة 6: الإطلاق (يوم 7+)

```
□ تفعيل المتجر على المنصة
□ اختبار استقبال طلب تجريبي
□ التأكد من ظهور المتجر في التطبيق
□ إرسال رسالة ترحيب للتاجر
□ جدولة متابعة بعد 3 أيام
```

## متابعة ما بعد التسجيل

### الأسبوع الأول

```
□ التواصل يومياً للتأكد من عدم وجود مشاكل
□ المساعدة في حل أي مشكلة تقنية
□ تشجيع التاجر على الترويج لوجوده على المنصة
```

### الشهر الأول

```
□ متابعة أسبوعية
□ مراجعة أداء المتجر (عدد الطلبات، التقييمات)
□ تقديم نصائح لتحسين الأداء
```
