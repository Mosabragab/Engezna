# Merchant Training Guide

**آخر تحديث:** يناير 2026

## أنظمة التشغيل الثلاثة

### 1. نظام القائمة (Standard Mode)

**مناسب لـ:** المطاعم، الكافيهات، المخابز

```
كيف يعمل:
┌─────────────────────────────────────────────┐
│ التاجر يضيف منتجاته ← العميل يختار ← يطلب │
└─────────────────────────────────────────────┘

المميزات:
✅ أسعار ثابتة ومحددة
✅ العميل يطلب مباشرة
✅ لا يحتاج تفاعل من التاجر قبل الطلب
```

### 2. نظام التسعير (Custom Mode)

**مناسب لـ:** السوبرماركت، الصيدليات، الخضار والفاكهة

```
كيف يعمل:
┌─────────────────────────────────────────────┐
│ العميل يكتب/يسجل/يصور طلبه                 │
│            ↓                                │
│ يرسله لـ 1-3 تجار في نفس الوقت             │
│            ↓                                │
│ كل تاجر يسعر الطلب بسعر اللحظة             │
│            ↓                                │
│ العميل يقارن ويختار الأفضل ✅               │
└─────────────────────────────────────────────┘

المميزات:
✅ لا يحتاج إنشاء قائمة منتجات
✅ لا يحتاج تحديث أسعار يومياً
✅ مرونة كاملة في التسعير
```

### 3. النظام الهجين (Hybrid Mode)

**مناسب لـ:** محلات تجمع بين النوعين

```
• قائمة منتجات للأصناف الثابتة
• نظام تسعير للطلبات المخصصة
```

## تدريب لوحة التحكم

### 1. استقبال الطلبات

```
الخطوات:
1. فتح صفحة "الطلبات" من القائمة الجانبية
2. الطلبات الجديدة تظهر بشارة "جديد"
3. الضغط على الطلب لرؤية التفاصيل
4. مراجعة المنتجات والكميات
5. تأكيد أو رفض الطلب

حالات الطلب:
• جديد (pending) → في انتظار التأكيد
• مؤكد (confirmed) → تم القبول
• قيد التحضير (preparing) → يتم تجهيزه
• جاهز للتوصيل (ready) → في انتظار السائق
• في الطريق (delivering) → السائق في الطريق
• تم التوصيل (delivered) → مكتمل
• ملغي (cancelled) → ملغي
```

### 2. إدارة القائمة

```
إضافة منتج جديد:
1. الذهاب إلى "المنتجات" → "إضافة منتج"
2. إدخال الاسم (عربي + إنجليزي)
3. تحديد الفئة
4. تحديد السعر ونوع التسعير
5. إضافة صورة المنتج
6. حفظ

أنواع التسعير:
• ثابت (fixed): سعر واحد
• بالوحدة (per_unit): سعر × الكمية
• متغيرات (variants): أحجام مختلفة (نصف كيلو، ربع كيلو)
```

### 3. استيراد من Excel

```
خطوات الاستيراد:
1. تحميل قالب Excel من لوحة التحكم
2. ملء البيانات:
   • اسم المنتج (عربي)
   • اسم المنتج (إنجليزي)
   • الفئة
   • السعر
   • نوع التسعير
   • المتغيرات (لو موجودة): نصف كيلو:480|ربع كيلو:250
3. رفع الملف
4. مراجعة البيانات
5. تأكيد الاستيراد

ملاحظة: يمكن استيراد حتى 500 منتج دفعة واحدة
```

### 4. التواصل مع العملاء

```
نظام الشات:
• كل طلب له محادثة خاصة
• يمكن إرسال رسائل نصية
• العميل يرى حالة "مقروء"
• الرد السريع يحسن التقييم
```

### 5. التقييمات

```
نظام التقييم:
• العميل يقيم بعد استلام الطلب
• التقييم من 1-5 نجوم
• يمكن للعميل كتابة تعليق
• التقييم يؤثر على ترتيب الظهور

نصائح لتحسين التقييم:
✅ الرد السريع على الطلبات
✅ جودة المنتجات
✅ التغليف الجيد
✅ الالتزام بالوقت
```

## نظام التسويات

### كيف تعمل التسويات

```
┌─────────────────────────────────────────────┐
│ طلبات الأسبوع                              │
│     ↓                                       │
│ حساب إجمالي المبيعات                        │
│     ↓                                       │
│ خصم عمولة إنجزنا (0-7%)                    │
│     ↓                                       │
│ التسوية (أسبوعياً)                         │
└─────────────────────────────────────────────┘

طرق الدفع:
• كاش
• تحويل بنكي
• InstaPay
• Vodafone Cash
```

### حساب العمولة

```
مثال (الموجة 1):
• أول 3 شهور: 0%
• بعد 3 شهور: 7%

مثال حسابي:
• مبيعات الأسبوع: 10,000 ج.م
• العمولة (7%): 700 ج.م
• صافي التاجر: 9,300 ج.م
```

| القناة | الاستخدام | وقت الرد |
|--------|-----------|----------|
| WhatsApp | مشاكل عاجلة | ≤ 30 دقيقة |

| المشكلة | الحل |
|---------|------|
| لا أستطيع تسجيل الدخول | إعادة تعيين كلمة المرور |

| لا أستطيع تعديل المنتج | التأكد من الصلاحيات |
| العميل لم يستلم | التواصل عبر الشات |
