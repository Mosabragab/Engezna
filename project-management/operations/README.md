## المؤشرات الرئيسية (KPIs)

### التجار

| المؤشر             | الهدف           | التكرار  |
| ------------------ | --------------- | -------- |
| التجار المسجلون    | 50-60/شهر       | شهري     |
| التجار المُدرَّبون | 90% من المسجلين | أسبوعي   |
| التجار النشطون     | 80% من المدربين | شهري     |
| وقت Onboarding     | ≤ 48 ساعة       | لكل تاجر |

### الجودة

| المؤشر               | الهدف           | التكرار  |
| -------------------- | --------------- | -------- |
| تقييم التجار         | ≥ 4.0           | شهري     |
| شكاوى العملاء        | ≤ 5% من الطلبات | أسبوعي   |
| وقت الرد على الشكاوى | ≤ 2 ساعة        | لكل شكوى |

### التوسع

| المؤشر            | الهدف        | التكرار   |
| ----------------- | ------------ | --------- |
| مدة إطلاق مدينة   | 4 شهور       | لكل مدينة |
| تكلفة إطلاق مدينة | ~156,000 ج.م | لكل مدينة |

## ملخص بالعربية

### النقاط الرئيسية

- **الغرض:** هذا الدليل يحتوي على جميع إجراءات تشغيل منصة إنجزنا
- **الهيكل:** يشمل أدلة ضم التجار، دعم العملاء، ضبط الجودة، والتوسع الجغرافي
- **المؤشرات الرئيسية للتجار:**
  - تسجيل 50-60 تاجر شهرياً
  - تدريب 90% من المسجلين
  - إبقاء 80% من المدربين نشطين
  - إتمام عملية الـ Onboarding خلال 48 ساعة
- **مؤشرات الجودة:**
  - الحفاظ على تقييم التجار ≥ 4.0
  - إبقاء الشكاوى ≤ 5% من الطلبات
  - الرد على الشكاوى خلال ساعتين
- **التوسع:**
  - مدة إطلاق مدينة جديدة: 4 شهور
  - تكلفة الإطلاق: ~156,000 ج.م
- **الفئات المستهدفة:** المطاعم، الصيدليات، السوبر ماركت، الخضراوات والفاكهه، البن والحلويات
